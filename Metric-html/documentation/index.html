Overview & Features


Metric - Responsive Multi-Purpose HTML Template

Metric is a modern, clean and professional HTML Template, It is fully responsive, it looks stunning on all types of screens and devices.

Template Features

  • HTML5 / CSS 3
  • Fully Responsive (Tested on Multiple Devices)
  • Full Width Layout
  • Portfolio page
  • Cloud zoom
  • Multi-options for Home Page
  • Inner Pages: shop grid, shop grid_left sidebar, shop list, single product, single product left sidebar, single product right sidebar, shopping cart, Checkout, Wishlist, Compare, Login, Blog, Detail Blog, About, Contact, shortcode ,...
  • Mega Top Menu
  • Mega Vertical Menu
  • Sticky Menu
  • Brand Slider
  • Hot Categories
  • Portfolio Page
  • HTML files
  • List & Grid View (Products)
  • Unique & Modern Design
  • Valid HTML5 Markup
  • Owl Carousel Slider With CSS3 Transitions
  • Free Google web fonts
  • Free Icon Fonts (Font Awesome)
  • Customizable & Organized
  • 404 page error
  • Compatible Browsers: IE9+, Firefox, Safari, Opera, Chrome
  • Included documentation.
